# Organizador de Arquivos em Python

Esse projeto foi feito por mim para praticar Python na parte de automação.

A ideia é simples: organizar arquivos automaticamente em pastas de acordo com o tipo (imagem, documento, vídeo, etc).

## O que o programa faz

- Cria pastas automaticamente (Imagens, Documentos, Vídeos, Outros)
- Identifica o tipo dos arquivos
- Move cada arquivo para a pasta correta

## Tecnologias usadas

- Python
- Biblioteca os
- Biblioteca shutil
